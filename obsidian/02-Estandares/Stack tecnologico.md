# Stack tecnologico

## Runtime y lenguaje

- Node.js 20+
- TypeScript 5.x
- ESM nativo

## Backend

- Fastify para la API
- Zod para validacion
- dotenv para configuracion local

## CLI y tooling

- tsx para ejecutar TypeScript en desarrollo
- ESLint para lint
- Prettier para formato
- pino y pino-pretty para logging

## Testing

- Vitest para unit e integration
- Supertest para pruebas HTTP
- Estructura lista para e2e en tests/e2e

## Deploy

- Vercel como configuracion de deploy actual

## Memoria y agentes

- .claude/ contiene agentes, comandos, hooks, reglas y skills
- obsidian/ funciona como memoria extendida del proyecto
- Hermes Agent consume este vault via claudio hermes optimize
